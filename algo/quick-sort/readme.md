# Быстрая сортировка (QuickSort)

Реализуйте функцию `QuickSort(data, comp)`, которая сортирует элементы вектора `data` на месте, используя алгоритм быстрой сортировки. 

Функция должна поддерживать кастомные компараторы. Если компаратор не передан, элементы должны сортироваться по возрастанию (с использованием `std::less`).

По сути тут две подзадачи:
1. Общая реализация, поддерживающая любые типы
2. [Специализация шаблонов](https://en.cppreference.com/cpp/language/template_specialization) для сортировки `int`-ов

Начните со второй.
Тесты только для специализации для интов можно вызвать через `./test_quicksort ~[advanced]`

**Требования и подсказки:**
1. Алгоритм должен корректно работать на вырожденных входах: пустой вектор, один элемент, все элементы равны, уже отсортированный или отсортированный по убыванию массив.
2. Избегайте деградации сложности до $O(N^2)$ на отсортированных данных и данных, состоящих из одинаковых элементов (подумайте над выбором опорного элемента и схемой разбиения, например, схемой Хоара или разбиением на 3 части).
3. Реализация шаблонной функции должна находиться в файле `quicksort.h`.
4. Ваша реализация должна корректно работать с типами, которые запрещено копировать *(move-only)*. Обратите внимание на то, как вы передаёте и меняете местами эелементы внутри массива.

В стандартной библиотеке сортировка (да и вообще многие алгоритмы) работает на итераторах.
